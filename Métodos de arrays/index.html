Nota: Para ver el resultado de cada uno de los métodos, presione F12

Probando el método shift

Esta función se encarga de eliminar el primer elemento de un array y devuelve ese elemento eliminado. Modifica el array original reduciendo la longitud de este (1 menos).

Probando el método unshift

Es el opuesto del método shift, permitiendo agregar uno o más elementos al inicio del arreglo, modificando el arreglo original.

Probando el método splice

Permite agregar, eliminar o reemplazar elementos en un arreglo. Modifica el arreglo original

Probando el método sort

Sirve para ordenar listas de palabras o nombres alfabéticamente, así como ordenar números de menor a mayor o de mayor a menor, modifica directamente al array original.

Probando el método slice

Sirve para extraer una parte del arreglo original, generando uno nuevo. No modifica el arreglo original.

Probando el método reduce

Toma el arreglo y lo reduce a un solo valor, aplica una función acumuladora a cada elemento, puede usarse para sumar números, contar elementos, concatenar cadenas, combinar objetos,etc.

Probando el método push y lenght

Push: Permite agregar uno o más elementos al final de un array, modificando el array original.

Lenght: Sirve para conocer cuantos elementos tiene un array, aunque también puede acortar un array de forma rápida.

Probando el método pop

Sirve para eliminar el último elemento de un arreglo, modificando directamente el array original y el tamaño del mismo.

Probando el método map

Sirve para modificar los elementos de un array, a diferencia del forEach, este método si genera un nuevo arreglo con los elementos transformados. No modificia el array original.

Probando el método lastindexof

Sirve para determinar la posición de la última aparición de un elemento en el array, en caso de que no exista, devuelve -1

Probando el método join

Funciona como un separador, convierte los elementos de un arreglo en una cadena de texto, separandolos con un delimitador específico.

Probando el método indexof

Sirve para encontrar y determinar la posición de un elemento específico que se encuentre dentro del array, en caso de no hacerlo, retorna -1.

Probando el método ForEach

Usado para recorrer un array y ejecutar una función en cada uno de los elementos, puede imprimir valores en consola, modificar variables externas.

Probando el método Find y FindIndex

Find: Recorre el arreglo y devuelve el primer elemento que cumpla con una condición determinada, en caso de que ninguno lo haga, retorna undefined.

FindIndex: Recorre el arreglo, y devuelve la posición del primer elemento que cumpla con una conidición determinada, si ninguno cumple retorna -1.

Probando el método Filter e Includes

Filter: Usado para crear un nuevo array con todos aquellos elementos que cumplan con una condición específica, a diferencia de muchos métodos, no modifica el array original.

Includes: Sirve para verificar si un array o una cadena de texto contiene un determinado valor.

Probando el método concat y reverse

Concat: Sirve para unir dos o más arrays, generando un nuevo array con los elementos de ambos sin necesidad de modificar los arrays originales.

Reverse: Sirve para invertir el orden de los elementos de un array, modificando directamente el orden del array original
